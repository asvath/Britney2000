# Britney2000
Britney2000 is a robot that will be able to map its surroundings, avoid obstacles and be able to reach a set destination.
## Specs

- 4 x80mm Mecanum wheels 
- 4 x 12V DC motors (330rpm)
- Mega2560 development board
- Lidar
- Raspberry Pi 3B+
